### CÓDIGOS

#### Código dentro de linea:

- Para volver al directorio anterior en la shell ejecuta el
  comando `cd ..`

- Para abrir un fichero con Vi escribe
  `vi ~/at/nombre_fichero.md`

#### Bloques de código

- Para escribir un **bloque de código** debemos usar la *comilla
  invertida* tres veces:

``` python
#!/usr/bin/env python3

a = 3
b = 2

print(a + b)
```
